## 面试须知：

> 电话面试的准备: 
> 1. 安静的地方(咖啡馆之类)
> 2. 带耳麦通话的耳塞
> 3. 笔和简历(方便应对面试官的问题) 
 
> 技术面试的准备:
> 1. 笔和简历(方便手写面试官的问题)
